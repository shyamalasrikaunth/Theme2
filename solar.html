Grid -Tied Solar system

The most popular options for all grid consumers. In which the Solar Panels generate DC Power which is converted to AC power using Grid tied Inverter. The generated power will be used for captive consumption and excess power will be fed into utility grid using net / gross meter.

Off-Grid System

Off-grid systems work independently of the power grid but have batteries which can store the solar power generated by the system.

Hybrid System

This is the middle ground option between grid-tied and off-grid. This has a has an energy storage component, like most off-grid systems, but maintains a connection to the power grid.

The system operates on power generated using solar PV (photovoltaic) system.

The photovoltaic array converts the solar energy into electricity, which is used for running the motor pump set.

The pumping system draws water from the open well, bore well, stream, pond, canal etc.
